# クイックスタートガイド

## 1. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで以下にアクセス：
```
http://localhost:3000
```

## 2. 基本的な使い方

### 顧客側

1. **トップページ**で「お客様用」をクリック
2. **テーブル番号**を入力（例: 1）
3. **番号入力**でメニュー番号を入力（例: 001）
4. 「メニューを追加」をクリック
5. カートに追加されたメニューを確認
6. 「カートを見る」をクリック
7. 「注文を確定する」をクリック
8. 注文確認ページで注文番号を確認
9. 「注文状況を確認」で注文の進行状況を確認

### 店舗側

1. **トップページ**で「店舗スタッフ用」をクリック
2. **ログイン**（任意のユーザー名・パスワードでOK）
3. **注文管理ページ**で注文一覧を確認
4. 注文のステータスを更新：
   - 「受付」→ 受付済み
   - 「調理開始」→ 調理中
   - 「完成」→ 完成

## 3. API接続の確認

開発サーバー起動後、ブラウザの開発者ツール（F12）で以下を確認：

- **Network**タブ: APIリクエストが成功しているか
- **Console**タブ: エラーが発生していないか

## 4. トラブルシューティング

### メニューが表示されない

1. ブラウザで直接APIにアクセス：
   ```
   http://mameq.xsrv.jp/radish/api/menus
   ```
2. JSONが返ることを確認
3. ブラウザの開発者ツールでエラーを確認

### 注文が作成できない

1. テーブル番号が入力されているか確認
2. カートにメニューが追加されているか確認
3. ブラウザの開発者ツールでエラーを確認
